# ALKING STORE

منصة حديثة لتحميل وبيع السكريبتات والملفات

## المميزات

✅ تسجيل الدخول والتسجيل الآمن
✅ تحميل وإدارة الملفات
✅ نظام الترخيص الفريد
✅ لوحة تحكم احترافية
✅ واجهة مستخدم عصرية

## المتطلبات

- Node.js 14+
- npm أو yarn

## التثبيت

```bash
# تثبيت المتطلبات
npm install

# تشغيل الخادم
npm start

# للتطوير (مع nodemon)
npm run dev
```

## الاستخدام

1. افتح المتصفح على `http://localhost:5000`
2. سجل حساب جديد
3. سجّل الدخول
4. انتقل إلى لوحة التحكم
5. قم برفع السكريبتات والملفات
